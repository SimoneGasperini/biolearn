Hopfield
--------

**Hopfield class** : 
it inherits from `Base` class and implements the unsupervised neural network
based on the Hopfield model.
Notice that it has some specific parameters to set the strength of the anti-hebbian learning
and to customize some of the model hyper-parameters.
